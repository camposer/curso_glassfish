Cómo configurar proyecto empresarial en Eclipse con Glassfish
=============================================================

Versiones
- Glassfish 4.1
- Eclipse Luna
- JDK 8

0.- Instalar JDK

1.- Descargar Eclipse Luna para JEE
https://www.eclipse.org/downloads/packages/eclipse-ide-java-ee-developers/lunasr1

2.- Instalar plugin de Glassfish para Eclipse
http://marketplace.eclipse.org/content/glassfish-tools-luna

3.- Descargar glassfish
http://dlc.sun.com.edgesuite.net/glassfish/4.1/release/glassfish-4.1.zip

4.- Configurar servidor dentro del eclipse. En la vista Servers dentro de la perspectiva Java EE, crear nuevo servidor Glassfish y especificar ruta, por ejemplo: C:/Users/Rodolfo/glassfish4/glassfish
